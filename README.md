# developerdk-random
## @Author developerdk
----

>```
> const { random } = require('developerdk-random');
> const randomimage = new random();
>```
**Package installation!**
----

```
 console.log(randomimage.porngif());
```
**Return 1 random image!**
----

```
 let images = 10;
 for (let i = 0; i < images; ++i) {
 	console.log(randomimage.porngif());
 }
```
**Return 10 random images!**
